---
title: Overview
description: Craftless is Browserless-style automation infrastructure for real Minecraft Java clients.
---

Craftless launches or attaches to real Minecraft Java clients, loads a small
in-client driver, discovers the live runtime capability graph, and exposes local
OpenAPI contracts for agents, tools, tests, and CI.

## Use The API Reference

The API reference is generated from the same Craftless-owned OpenAPI metadata
used by the daemon and adaptive CLI. Start with the supervisor API, then fetch a
client's generated API at runtime for gameplay actions and resources.
